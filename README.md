# Monopolly
This project is a monopoly game created for course work for Basics of Programming discipline. The main goal of project is to use programming patterns.

For this project Unity was used.

Used programming patterns:
- Strategy
- Command
- Chain of responsibility
- Observer
- Facade
- Singleton
- Builder
- Factory

Mechanics, rules and board layout are based on existing board game. Non-commercial, for learning purposes only.
